### 5. Remove Fast Mode

Fast mode is Opus 4.6 only. Remove entirely:

- `ENABLE_FAST_MODE` valve
- `ENABLE_MY_FAST_MODE` user valve
- `_resolve_fast_mode()` method
- Fast mode beta header in `_get_headers()`
- `payload["speed"] = "fast"` in `_prepare_payload()`
- `is_fast_mode` parameter from `_format_token_usage()`
- `is_fast_mode` variable in `pipe()`, `stream_response()`, `non_stream_response()`
- `price_mult` logic in cost estimation
